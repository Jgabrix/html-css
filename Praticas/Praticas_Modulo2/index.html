Estrutura básica do HTML

O HTML5 conhecido como HyperText Markup Language, o que significa Linguagem de marcação de Hipertexto, tem como responsabilidade na criação de qualquer site a criação do conteúdo.
Dessa forma, quando inicia-se o desenvolvimento do HTML, é necessário o uso de tags semânticas, que em outras palavras, tem como foco no sentido das palavras/texto.

Exemplo da estrutura básica: 

     
         DOCTAYPE html
         html
         header
             meta charset = "UTF-8"
             meta name="viewport" content="width=device-width, initial-scale=1.0"
             title document title
         body body
         html
