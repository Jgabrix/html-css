Título

No HTML5 existem tags que funcionam de forma semântica e não como forma, pois a forma ou melhor dizendo o estilo são usadas na parte de CSS3.

Tags semãnticas

São as tags que dão o sentido a palavra, por exemplo:
strong: deixa qualquer palavra em negrito
mark: funciona como marca texto
em: deixa qualquer palavra em itálico para dar ênfase
small: serve para diminuir a palavra para dar importância a ela
ins: sublinha o termo
sup: deixa sobrescrito; x2 + 4
sub: deixa subscrito; H2O
address: refere-se ao endereço:

condomino pedra caída

são tags semânticas que dão significados as palavras do texto em HTML, pois essa é uma linguagem de marcação, de conteúdo.
