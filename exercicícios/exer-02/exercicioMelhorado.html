Html: fundamentos

O Html é uma linguagem de marçação que tem como funcionalidade mostrar conteúdos por meio de Tags.
Dessa forma é necessário ter treinos consistentes para aprender o máximo de tags possíveis.

#Tags

* <br> quebra de linha.
* <p> faz um parágrafo.
* <h1> cria um título.
* <hr> cria uma linha.

#Símbolos

Símbolos no html são espécificos e só é possível aprende-los através da prática.

* ®
* ©
* ™
* €

#Emojis

Para colocar emojis no html é necessário o código do emoji e utilizar &#x"colocar o código do emoji sem o (u+)"
exemplo:
🖖 code: U+1F596
